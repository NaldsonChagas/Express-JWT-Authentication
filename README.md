# Express-JWT-Authentication
A simple Authentication with JWT using Node.Js and Express

## Avaible routes


* "/" post: Only for login
* "/user" post: create a user

* "/user" get: List all user (require a token for access)


## Default user: 

| username  | password  |
|---|-------------------|
|   admin   |    123    |

## Libraries

* express 
* bcrypt
* express-kun
* jsonwebtoken
* sequelize
* sqlite3
